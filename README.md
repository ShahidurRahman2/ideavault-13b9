IdeaVault – Startup Idea Sharing Platform

Live Site: https://ideavault-13b9.vercel.app

Project Overview

IdeaVault is a modern startup idea sharing platform where users can explore innovative ideas, share their own concepts, interact through comments, and discover trending startup solutions from different categories like AI, Health, Education, Technology, and Productivity.

Features
User authentication with Email/Password and Google Login
Add, update, and delete startup ideas
Trending ideas section on homepage
Search and filter ideas by category
Comment system with edit and delete functionality
Responsive design for mobile, tablet, and desktop
Dark and Light theme support
Dynamic banners with smooth slider
Private routes for secured pages
Modern UI/UX with Tailwind CSS
